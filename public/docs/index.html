Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Handle user login

Example request:
curl --request POST \
    "http://localhost/api/auth/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"feeney.zachery@example.net\",
    \"password\": \">Z?bUng}k+vvsn|BRi(Y\"
}"

const url = new URL(
    "http://localhost/api/auth/login"
);

const headers = {
    "Content-Type": "application/json",

};

    "email": "feeney.zachery@example.net",

};

Handle user registration with medical facility support

Example request:
curl --request POST \
    "http://localhost/api/auth/register" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=yjjonykvaljnpnnuuycvllrvgouuhlxjrdnwjpsfcdpgvrjoutiuxey"\
    --form "email=wschultz@example.com"\
    --form "password=M,Q@1)nh;irt:Ko==F!}"\
    --form "user_type=medical_facility"\
    --form "whatsapp_number=minima"\
    --form "image=@/tmp/phpggvam6t2v19f1vp0Vob" 
const url = new URL(
    "http://localhost/api/auth/register"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('name', 'yjjonykvaljnpnnuuycvllrvgouuhlxjrdnwjpsfcdpgvrjoutiuxey');
body.append('email', 'wschultz@example.com');
body.append('password', 'M,Q@1)nh;irt:Ko==F!}');
body.append('user_type', 'medical_facility');
body.append('whatsapp_number', 'minima');
body.append('image', document.querySelector('input[name="image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

The id of an existing record in the units table.
